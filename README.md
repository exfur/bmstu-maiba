# 🚀 Методы и инструменты в бизнес-анализе (МИИБА)

Мастер-репозиторий учебных материалов курса для направления «Бизнес-информатика». Содержит интерактивные семинары, шаблоны для сквозного курсового проекта и контекст для персонального AI-тьютора.

## 🧰 Шаг 0. Подготовка (Установка базового софта)
Перед началом работы убедитесь, что на вашем компьютере установлены всего две программы:
1. **[Visual Studio Code (VS Code)](https://code.visualstudio.com/)** — наша рабочая среда разработки.
2. **[Git](https://git-scm.com/downloads)** — для скачивания материалов курса.

---

## 📥 Шаг 1. Клонирование материалов курса

Скачайте проект на свой компьютер. Для этого создайте учебную папку (например, `C:\Projects`), откройте в ней терминал (или Git Bash) и выполните команду:
```bash
git clone https://github.com/exfur/bmstu-maiba.git
```

После завершения загрузки у вас появится рабочая папка `bmstu-maiba`.

---

## ⚡ Шаг 2. Первичный запуск и ультра-быстрая автонастройка

Мы используем современный менеджер пакетов `uv`. Он работает на Rust, летает в 10-100 раз быстрее стандартного `pip` и самостоятельно управляет изолированными версиями Python, не засоряя вашу операционную систему.

1. Откройте **VS Code**.
2. Перейдите в меню `File` -> `Open Folder...` (Файл -> Открыть папку) и выберите скачанную папку `bmstu-maiba`.
3. Откройте встроенный терминал VS Code (нажмите комбинацию клавиш `Ctrl + ~` или в верхнем меню выберите `Terminal` -> `New Terminal`).
4. Скопируйте в терминал следующую команду для установки `uv` и нажмите `Enter`:
```powershell
powershell -ExecutionPolicy ByPass -c "irm [https://astral.sh/uv/install.ps1](https://astral.sh/uv/install.ps1) | iex"
```


5. **Важно:** Закройте текущий терминал (нажмите на значок корзины справа внизу на панели терминала) и откройте новый через `Terminal` -> `New Terminal`.

> ⚠️ **Магия автоматизации:** Как только вы открыли новый терминал в папке проекта, VS Code автоматически распознает файл конфигурации и запустит команду `uv sync` в фоновом режиме. Система сама создаст изолированное окружение `.venv` и скачает правильную версию Python и все библиотеки курса (pandas, pycaret, shap, prophet и др.). Вам больше ничего не нужно устанавливать вручную!

## 🧠 Шаг 3. Подключение персонального AI-тьютора

Если в процессе выполнения практических заданий или курсового проекта вы столкнулись с непонятной ошибкой или застряли на написании кода — не паникуйте. У вас есть персональный ментор.

1. Зайдите в папку `seminars/` и найдите там файл **`help_me.md`**.
2. Скачайте его и отправьте первым сообщением в любой ИИ-ассистент (ChatGPT, Claude, Deepseek и т.д.).
3. Нейросеть мгновенно превратится в умного, поддерживающего тьютора по нашему курсу. По правилам игры она **не будет писать код за вас**, но с помощью наводящих вопросов поможет исправить любой баг.
4. В каждом задании семинара вы найдете специальные **хэштеги-закладки** (например, `#SEM1_TASK3_BUG`). Скидывайте их ассистенту вместе со своим кодом, чтобы он сразу понял контекст проблемы.
