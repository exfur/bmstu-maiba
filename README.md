## 🧰 Шаг 0. Скачиваем базовые программы

Перед началом работы нам понадобятся всего две программы.

1. **Visual Studio Code (VS Code)** — это наша главная рабочая среда, продвинутый "блокнот" для кода.
* Перейдите на официальный сайт: [code.visualstudio.com](https://code.visualstudio.com/).
* Нажмите на большую синюю кнопку **"Download"** по центру экрана.
* Запустите скачанный файл и устанавливайте программу, просто нажимая "Далее" (оставьте все настройки по умолчанию).


2. **Git** — специальная утилита, которая позволит безопасно и быстро скопировать все материалы курса к вам на компьютер.
* Перейдите на сайт: [git-scm.com/downloads](https://git-scm.com/downloads).
* Скачайте версию для вашей операционной системы (Windows или Mac).
* Запустите установщик и всё время нажимайте "Next", ничего не меняя.

## 📥 Шаг 1. Копируем материалы курса на свой компьютер

Теперь нужно забрать папку с учебными материалами.

1. Создайте обычную папку на вашем компьютере (например, на диске `C` создайте папку с названием `Projects`).
2. Откройте эту папку.
3. Щелкните правой кнопкой мыши по пустому белому фону внутри папки и в появившемся меню выберите **"Open Git Bash here"** (Открыть Git Bash здесь). У вас откроется черное окно.
4. Скопируйте команду ниже:
```bash
git clone https://github.com/exfur/bmstu-maiba.git
```


5. Вставьте её в открытое черное окно (используйте клик правой кнопкой мыши -> **Paste** или комбинацию `Shift + Insert`) и нажмите клавишу `Enter`.
6. Дождитесь загрузки файлов. После этого черное окно можно закрыть — внутри вашей папки появится новая папка `bmstu-maiba`.

## ⚡ Шаг 2. Открываем проект и запускаем "Магию"

Чтобы не устанавливать кучу технических библиотек вручную, мы воспользуемся умным менеджером `uv`, который всё сделает за вас.

1. Откройте установленную программу **Visual Studio Code**.
2. В самом верхнем левом углу экрана найдите слово **File** (Файл) и нажмите на него. В открывшемся списке выберите пункт **Open Folder...** (Открыть папку...).
3. Найдите только что скачанную папку `bmstu-maiba`, выделите её и нажмите кнопку "Выбрать папку" (Select Folder).
4. Снова посмотрите на самую верхнюю панель меню программы. Найдите там вкладку **Terminal** (Терминал), нажмите на неё и выберите **New Terminal** (Новый терминал). (Или нажмите одновременно клавиши `Ctrl` и `~`).
5. В нижней части экрана откроется новая панель. Скопируйте команду ниже, вставьте её в эту нижнюю панель и нажмите `Enter`:
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

```


6. **Критически важный шаг:** Обратите внимание на правый край этой нижней панели терминала. Найдите там маленькую иконку **мусорной корзины** 🗑️ и нажмите на неё, чтобы закрыть текущий терминал.
7. Снова откройте новый терминал: в самом верхнем меню выберите **Terminal** -> **New Terminal**.
8. Готово! Программа сама "увидит" настройки и в фоновом режиме скачает нужную версию языка Python, а также все необходимые библиотеки (pandas, pycaret, shap и др.).

## 🧠 Шаг 3. Как получить помощь, если вы застряли

Если при выполнении заданий вы столкнетесь с ошибкой — не паникуйте. Для вас подготовлен персональный AI-помощник.

1. Посмотрите в левую часть экрана VS Code (там находится панель со списком файлов). Найдите папку `seminars/` и нажмите на неё, чтобы она раскрылась.
2. Внутри вы увидите файл под названием `help_me.md`.
3. Отправьте его самым первым сообщением в ваш любимый ИИ (ChatGPT, Claude или Deepseek).
4. Теперь нейросеть знает все правила нашего курса. Она превратится в вашего репетитора, который всегда поможет и подскажет с любым заданиме курса.
5. Во время решения задач вы будете встречать в тексте специальные "теги" (например, `#SEM1_TASK3_BUG`). Просто отправляйте нейросети свой проблемный код вместе с этим тегом, чтобы она моментально поняла, с каким именно заданием вы бьетесь.
